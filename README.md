# TextReader

A simple Python text-to-speech program using `pyttsx3`.  
This program reads aloud any text you enter.

## How to use

1. Install Python 3.x and `pyttsx3` library:
   ```bash
   pip install pyttsx3
2. Run the program:
    python tts.py
3. Enter the text you want to hear, and your computer will speak it.

Features
-Converts text to speech
-Works offline
-Simple Python script
